# 英语词汇学习管理系统 - 规格说明书

## 1. 项目概述

- **项目名称**: VocabVault - 英语词汇宝库
- **项目类型**: 单页 Web 应用
- **核心功能**: 查询英语单词/短语并获取释义、发音、例句，支持词汇收藏、标签管理和智能检索
- **目标用户**: 英语学习者，尤其是商务专业人士

## 2. UI/UX 规格

### 布局结构

- 顶部导航栏（固定高度 64px）
- 主内容区：左侧侧边栏（280px）+ 右侧内容区
- 响应式断点：移动端 < 768px，平板 768–1024px，桌面 > 1024px

### 视觉设计

- **主色**: #2D5A4A（深松绿）
- **次色**: #F5F0E8（暖米白）
- **强调色**: #D4A853（古铜金）
- **字体**: Playfair Display（标题）、Source Sans 3（正文）、JetBrains Mono（音标）

### 核心组件

- 搜索栏：输入框 + 搜索按钮，支持回车搜索
- 单词详情卡片：单词、音标、发音按钮、词性、释义、例句、收藏按钮、标签
- 词汇列表：网格/列表视图，支持按标签筛选、排序、搜索

## 3. 功能规格

### 3.1 单词查询

- 使用 Free Dictionary API 获取单词数据（优先）
- 短语或未命中时使用 MiniMax 等 LLM 补充
- 展示：拼写、IPA 音标、发音音频、词性、释义、例句、同义词等

### 3.2 词汇收藏

- 星标收藏，记录收藏时间、单词及查询结果
- 支持自定义标签与编辑

### 3.3 标签系统

- 预设：难度（简单/中等/困难）、词性（名词/动词/形容词/副词等）、来源（日常/商务/学术/考试）
- 支持自定义标签

### 3.4 筛选与搜索

- 按标签多选筛选、关键词模糊搜索、排序（日期/字母等）、视图切换

### 3.5 数据持久化

- 使用 LocalStorage 存储词汇与设置

## 4. 验收标准

- 页面风格统一、响应式正常
- 单词查询返回完整信息，发音可播放
- 收藏、标签、筛选、LocalStorage 持久化正常
- 搜索回车触发、收藏/删除有反馈、空状态有引导
